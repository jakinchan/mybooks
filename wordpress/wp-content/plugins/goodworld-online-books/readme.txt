PDF bookshelves for WordPress + SQLite + Free PDF to Flipbook.

== Description ==

GoodWorld Online Books adds an online book custom post type, bookshelf shortcodes, access control, and templates for displaying Free PDF to Flipbook shortcodes.

== Dependencies ==

* SQLite Database Integration
* Free PDF to Flipbook

Missing dependencies are shown as admin notices and do not cause fatal errors.
